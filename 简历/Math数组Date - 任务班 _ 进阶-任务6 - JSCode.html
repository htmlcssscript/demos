6.数组方法里push、pop、shift、unshift、join、split分别是什么作用？用 splice函数分别实现push、pop、shift、unshift方法.

答:首先，数组里貌似没有split()这个内置方法，应该说的是join()吧...?

  1. 其次，push、对应的是pop、shift对应的是unshift其中push会在原来数组的基础上添加一个新的成员至数组的末尾，而pop则返回数组去处的最后一个成员的值,这些方法都将改变数组并形成我门常说的后进先出的栈方法，同样的shift会删除并返回数组的第一位成员，并修改原数组，而unshift就会在数组的第一位添加一个新的成员，并返回修改后的数组。

  2. 关于，join和split其实是比较对立的两个方法，join();是用于将一个数组以某种特定的分隔符转换成字符串，比如:join(''),不添加任何分隔符,join('|')或join(','),以"|"或","逗号添加分隔符,并返回字符串。

  3. 最后，关于split其实就是以相应的分隔符来划分界限并返回相应的数组和数组成员，其用法是split();返回的是一位数组成员但包括所有的字符串，split(''),空字符串则均等分割每一个字符串包括逗号,以此类推.

	var nowTime = Date.now(); 
	var endTime = Date.parse(str);
	var gap = endTime - nowTime;
	var days = parseInt(gap / (1000 * 60 * 60 * 24));

	var minutes = parseInt(((gap % (1000 * 60 * 60 * 24)) % (1000 * 60 * 60)) / (1000 * 60));

	return "距离除夕还有" + days + "天" + hours + "小时" + minutes + "分钟" + seconds + "秒";
